時間Sat Dec 20 11:27:00 2014
1.媒體來源:UDN 2.完整新聞標題: 桃園／鐵路地下化 鄭文燦：政策選擇無關錢 3.完整新聞內文: 【聯合報╱記者鄭國樑／桃園報導】 2014.12.19 03:20 am 桃園直轄市長當選人鄭文燦強調鐵路地下化只是「政策選擇」，即使要增加500億元工程 費還是要做，而且由中央負擔經費，因為其他直轄市都不需要負擔鐵路工程經費，桃園鐵 路高架化沒道理要地方負擔120億元，不能把桃園當成「次等公民」。 鄭文燦表示，他要讓中央知道桃園鐵路地下化是唯一選項，「錢和工期不是討論重點」。 他說，工期如果要多3年、工程費用增加500億還是選擇地下化，中央要明白所有鐵路是國 家服務不是地方服務，沒有道理要求地方編列。 「希望桃園6位國民黨籍立委支持鐵路地下化，」鄭文燦說，中央必須比照過去在直轄市 辦理鐵路工程的標準，由中央編預算而非地方。 鄭文燦指出，去年桃園繳稅1800億元，如果鐵路地下化需要增加500億元被視為不可能， 他要告訴大家，當年中山高五楊高架工程後共計追加600億元，桃園鐵路地下化只要500億 元。 他表示，如果用5年工期一年追加100億元，對照中央1年2兆元的預算而言，1年不給桃園 100億元預算說不過去，何況這攸關桃園百年建設、桃園人的需求心願。 現任交通局長高邦基獲留任，鄭文燦表示高是政務官，已經和他溝通過，「他很清楚我的 理念而且會努力實現」，何況鐵路局、台鐵工程局的人也說地下化是正確的，而且桃園地 質比台北好，台北是爛泥黑土，桃園挖3尺就是卵礫石層條件好，施工快可以分段同時進 行，就能解決時間的問題。 4.完整新聞連結 (或短網址): http://udn.com/NEWS/NATIONAL/NATS4/9139093.shtml 5.備註: 錢跟工期不是重點 那什麼才是重點 台灣沒有任何一個鐵路地下化工程是中央十成的好嗎..... 除了南鐵以外其餘都至少三成 不想出錢 那就別蓋了 別的縣市很想要 = = -- ─┬─ ─┬─╮ ╮ ╭╮ │◎ ◎│ ├─╮╭─╮│ ╱ ╭─╮ │ ╴╴ ╭─╮╭╮╭╮ │ ∪ │ │ │╭─┤├┤ ╭─┤╭┼╮ ╱ │ │ Ｖ │ ψ │ ﹀ │ │ ││ ││ ╲ │ │ │ ╳ │ │ │ │cito ╰───╯ ╯ ╰┴─┴╯ ╰┴─┴─┴─┴──┴─┴╯ ╰ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 36.237.195.13 ※ 文章網址: http://www.ptt.cc/bbs/Gossiping/M.1419046023.A.561.html
