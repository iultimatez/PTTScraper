標題[新聞] 五分車時代 大寮人揪團寫回來
時間Sun Dec 21 23:59:54 2014
五分車時代 大寮人揪團寫回來 大寮區公所廣邀民眾蒐集史料並訪談耆老，撰寫家鄉史，還原五分車時代的庶民文化，也 串起一群人本來互不認識的大寮人，因為寫書成立line群組，組「愛家園」社團，未來將 和大寮導覽員持續行銷大寮。 公所今年年中，透過在地社團、基金會、發展協會及各里長，發布招募社區寫手的消息， 吸引當地社區民眾、公所同仁、退休人員等加入。雖然多數沒相關經驗，但公所安排學習 編採、攝影及認識鐵路史和大寮史的課程學習。 擔任導覽員的寶愛回想寫書過程最困難的地方時，提到早期純樸農村忙農務，可說是「連 吃飯都沒時間，更不可能去拍照」。即便有人有照片，也需能說服當事人接受深度採訪並 提供照片，受訪者中還有早年開糖鐵的司機，實在非常難得。 公所員工簡玉妍（37歲）說，本身就是大寮人，但對大寮的了解很少，參與寫書多了解很 多在地史，現在經過自行車道時，腦海裡還會浮現訪談過程，及耆老描述的鐵道樣貌。 受訪者李坤珠阿嬤（83歲），年輕時常在農餘時分，到甘蔗田削蔗根仔，對早年甘蔗田的 大量人工作業的分工及糖鐵運輸量等仍印象深刻，直說「現在社會進步很多」。但講起當 初一起削蔗根仔的同伴也感嘆：「他們一個一個都不在了」。 大寮區長王宏榮表示，現在大寮的發展網絡就是糖廠五分車鐵道所經之處，五分車帶著大 寮成長，隨時代變遷，糖廠的經濟效益大不如前，鐵道荒廢，現在成為單車步道，透過寫 書讓大家再度找回記憶中的大寮，希望能延續這份力量。 「鐵支路五分車．追憶」完成前夕，志工們在凌晨5點熱血相約，實際走訪一遍大寮路段 的五分車，用接力方式介紹自己熟悉的糖鐵路段，讓大家都很難忘，不想因為書籍完成就 結束這段緣分，將繼續一起作促進大寮發展的工作。 【2014/12/21 聯合報】@ http://udn.com/ http://udn.com/NEWS/DOMESTIC/DOM6/9142495.shtml -- + ╱ █ ╲ ╱ █ ╲ ╱ █ ╲ + + /────\ /────\ /────\ + ( ′▽‵∩ (′▽‵∩) ( ′﹂‵) + (( (つ ノ (つ ノ (つ つ )) + ˋ ( ） ( ˋノ ) ) ) (__)じ し (__) (__) __) ψQSWEET -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 220.136.172.164 ※ 文章網址: http://www.ptt.cc/bbs/Gossiping/M.1419177598.A.68E.html
