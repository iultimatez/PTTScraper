我實在不懂很多人心中紅的定義是什麼。 請要知道一件事情，在日本，動畫公司不是吃飽沒事幹閒著要交業績的， 如果你原作書銷量不夠人氣不夠高，動畫公司才不願意花錢改動畫來賠自己錢。 雖然改了動畫之後反而被罵翻也是很多…… ※ 引述《kid725 (凱道基德)》之銘言： : ※ 引述《lifepass (乖乖)》之銘言： : : 有些漫畫家只紅一套漫畫 : : 像和月伸宏只紅神劍闖江湖、高橋和希只紅遊戲王 : : 想當初的聲勢應該不下於富奸、鳥山明吧 : : 怎會有如此淒涼的下場呢 : : 有沒有只紅一套漫畫的漫畫家的八卦 : 這兩個狀況不太一樣 : 高橋是遊戲王就賺不完 你看遊戲王幾個系列了 數到我懶得數 : 他自從遊戲王走紅後 遊戲王冷飯炒不完 數錢數到手軟 : 你要說這叫淒涼 全日本99%的漫畫家都要去上吊了 : 和月就真的只紅一套 之後被斬兩套 一套畫到現在沒人管他死活 : 這兩個作家表現出最大的差異 你把武裝鍊金擺哪去了…… 武裝鍊金當時被腰斬，是因為當時的編輯長剛上任， 而且更加執行問卷調查至上主義， 和月本人說過其實他很喜歡機械風和西洋風， 但當時難以被讀者接受還是被腰斬了。 只是沒想到真腰斬之後編輯部反而收到一片罵聲來信， 才讓和月在赤丸JUMP上畫完。 後來還出了動畫和遊戲。 這叫不紅我真的不知道什麼叫紅了。 : 就在於一部作品紅了 漫畫家有沒有能力繼續讓這部作品紅下去 : 當初遊戲王原本只是小格局的小屁孩通靈鏟奸除惡的故事 : (但老實說 我還滿喜歡這時期的故事) : 後來因為賣不好 只好轉型 作者花了一晚想出遊戲牌的規則後硬上 : 結果大紅 就抽牌決鬥到現在了 : 和月的狀況就不太一樣 神劍初期連載非常順利 : 但是京都篇後評價就開始下滑 (京都篇太成功了 結果難以超越) : 後來作者本來還打算讓他們跑到北海道去搞拓荒 : 但最後不被看好就只好把故事結束 : 可能有了大紅的經驗有自信了 覺得再新創一部作品也可以受到歡迎吧 : 就把沒用到的拓荒精神拿來創作"西部神槍手" 結局就是慘到被腰斬 之後就是現在這樣了 : (老實說我個人最喜歡和月在京都篇時的表現 : 後來他的畫風變得太有稜有角 已經沒辦法表現那種時代劇浪漫了) : 這邊的是兩種類型 畫一部吃一輩子的 跟畫一部後只能吃老本的 : 但老實說 大部分的當紅漫畫家都是屬於這兩者 : 現實是 在JUMP史上 畫了一部大紅後 還可以再大紅一部的 用十隻手指都數得出來 : 這種作家根本稀有到不行 : 比較具代表性的就是鳥山明跟富姦 : 很多人提到鳥山明 只知道七龍珠 但是當年真正讓他走紅的是"怪博士與機器娃娃" : 這部作品在當年紅到可以開主題樂園 成為很多人學習模仿的對象 : 畫火影的岸本齊史 小時後畫阿拉蕾結果被老媽塗口紅讓他生平第一次暴走 : 當時去投稿集英社的新人漫畫家 有一半是怪博士與機器娃娃的風格 : 由此可知這部作品當時到底有多紅 : 到最後鳥山明沒梗了 以承諾結束後會再創作一部作品為條件 編輯部才肯讓他完結 : 結果新接檔的七龍珠不受觀眾喜愛 人氣不振到差點就被腰斬 : 編輯部操刀讓悟空開快轉把紅衣軍秒掉 轉型格鬥漫畫才把七龍珠救回來 : 之後一直修正路線 才讓七龍珠成為日本史上最重要的漫畫之一 : 富姦的幽遊白書也不用說了 本來也只是勵志小品 : 結果一樣是人氣危機 硬凹成格鬥漫畫才救回來 : 但是最後富姦承受不住週刊連載的壓力崩潰 硬生生把幽遊白書自宮掉 : 之後的獵人才又大紅 富姦又大病病不完(官方說法 實際上就是擺爛) : 所以要能紅兩部作品 也真的不容易 : 其實很多時候是被出版社救回來 而不是只算作者的功勞 JUMP近年來最大的弊病就是問卷調查至上主義， 這已經被罵很久了。 幽白和七龍珠以及許多更早的經典作(城市獵人、魁男塾、聖鬥士星矢、……)， 正好是問卷調查取向沒有那麼嚴重的時候。 另外，幽白可不是人氣危機才轉成王道格鬥漫， 富姦一開始確實只想畫： 「一個不良少年是顧死亡經過一連串靈魂的冒險終於回到喜歡的女孩身邊」 這種小品，是因為當時問卷調查超級大紅人氣大爆發，加上富姦當時超菜， 沒資格講話大聲， 富姦才被編輯部要求繼續畫下去，單行本第三集才突然變成靈界偵探變成王道漫。 現在回頭看幽白，就知道這作品根本不能當少年漫…… (富姦的一路過程可是從被責編痛罵到現在責編得送高級壽司去巴結……) 問卷調查至上主義問題在哪？問題可大了！ 捫心自問，上了大學之後的鄉民們，就算你每個禮拜買寶島和新少快好了， 會每個禮拜把回函填好剪下來寄回去的有多少人？ 真正會填問卷寄回去的幾乎都是國高中生， 因此小屁孩愛看的「王道漫」幾乎成為不得不的主流。 但是就算是80年代的王道漫，也是需要鋪陳的， 但是小屁孩哪裡知道什麼叫劇情沉潛的鋪陳？哪裡知道什麼叫做見好就收？ 所以大家就看到了有潛力成為經典的作品因為問卷調查人氣太高而拖成糞作， (光速蒙面俠21) 或者是鋪陳沉潛到還沒拉起高潮就被腰斬。 (武裝鍊金) 但集英社沒想到的是，JUMP發行幾十年下來， 許多當年的屁孩現在都是大叔了， 即使是大叔也還是有許多人會持續看JUMP， (阿銀：「都這個年紀了我還是離不開JUMP啊……」) 因此以問卷調查來決定作品是否受歡迎的失真度非常高。 剛剛看到有人講安西信行只紅一部烈火之炎， 魔兵傳奇MARS也是很紅，只是因為其觀眾取向在小學生， 所以很多年紀較長的看不下去而已， 魔兵傳奇也是動畫和遊戲都有，還有別的作者用它的世界觀繼續畫下一部續作。 -- ▃▃▃▃
