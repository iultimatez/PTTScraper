如果各位網友實際去了解這些國營事業的歷史，很多裡面的經營都是從國民黨的黨營事業留下來的！接下來的內容都會有連結資料做佐證，請各位耐心看完！ : 我想首先從ㄧ些檯面上，記者追的到的訊息；有新聞記錄的講起！這樣比較不會淪為口水，而缺乏事實依據！ : 我們都知道，油、電、水這些都跟我們小老百姓的生活習習相關；台電、中油講好聽為國營事業；可是在台電、中油裡面有一些人事佈局和營運是立法院管不到卻有私底下在運作的！ : 我們常常講「肥貓」，但是這些所謂肥貓有沒有具體例子？我先以先前被民進黨立委罵的半死的天然氣為例子！ : （1）首先講天然氣 : 台灣天然氣市場已經變成了國民黨的黨營事業了！而這些天然氣公司已經變成國民黨籌傭的工具！如果還記得當出被民進黨立委罵的半死的退輔會（安排退休官員到天然氣公司當董事） : 以下我highlight 引用文章內容並節錄當中重點： : 台灣的天然瓦斯公司是幾近「壟斷」的「寡頭獨占」事業，和自來水事業差不多。而天然氣瓦斯（LNG,指管線運輸瓦斯而非桶裝）之所以會落入由國軍退除役官兵輔導委員會一手獨攬，要回溯民國五十年代末期。其時瓦斯（天然氣）燃料還是新興的能源，家庭用只占一小部分，大部分供工業使用，但其清潔、便宜及運輸方便，成本售價較低。行政院看準了這一點，居然在1968年發布行政命令：家庭用有管線輸送之天然氣，除原有（大台北）已有管線供氣者外，全交由退輔會經營。此一張薄紙之行政命令，就把全台灣之管線天然氣生意，全交由退輔會「寡占」。然退輔會手持 : 「尚方寶劍」，自1971年起從台北近郊的欣欣天然氣公司著手，透過黨政系統尋找民間「合作者」，共同投資設立「欣」字輩天然瓦斯公司，針有欣欣、欣基、欣桃、欣中、欣林、欣彰、欣雲、欣南、欣雄、欣屏、欣湖等共十一家。可說除大台北瓦斯以外，全是欣家「兄弟」的天下。 : 妙的是，行政院的行政命令規定「由退輔會經營」，但退輔會卻是透過黨政系統，找各縣市的「財團椿腳」共同投資，退輔會股份不超過49%，以免成為國營事業，要受立法院監督。（此點應有違反法令之處，退輔會持股不到一半以上，名目上怎能構成＜經營＞要件？）另方面，控股49%，事實上卻可做到控制作用，從董事長、董監事到總經理等高層管理階層，皆由退輔會指派。欣字輩天然氣公司成為退輔會「禁臠」，也就是安排一些退將的好去處，外行領導內行，績效放一邊。而參與投資合作的民間股東，是黨國的御用商人，也能從旁利用特權分一杯羮，皆大歡喜。這就是 : 康寧祥立委在立法院揭發的「黨國資本勾結」的變體，對台灣政治發展也有不利影響。 : 康寧祥在1975年國營預算審查時，發現了國民黨營「瑞華公司」壟斷中油液化瓦斯（LPG,又稱桶裝液化瓦斯）的供應，平白每公斤「抽佣」一元，（即中油每售一公斤液化瓦斯，得8.5元，瑞華公司以總經銷之名，從中抽取1元佣金）。他即在立法院質詢，要求不能由黨營事業霸占且「中間剝削」，要求改由中油直接經營躉售，利益歸於全民，糾正「國庫通黨庫」的歪風。伊時，台灣每年需桶裝（鋼瓶）液化瓦斯45萬公噸，黨營瑞華公司只共投資了二億多元設備，一年就可淨賺4億5千萬元，是筆不得了的龐大數字。如果現在還存在此「抽佣」制度，因桶裝液化瓦斯用戶達320렊 : U戶，國民黨一年可不勞而獲約10億的暴利！若比照當時比例「抽佣」，則每年可獲40億元以上。 : 資料來源： : http://twtext.blogspot.de/2014/08/servicemen.html : （2）在講中油 : 中油工會的理事長莊爵安本身就是國民黨的中常委！同樣是全國產業總工會理事！這是一個國民黨利用黨政平台把手伸進工會控制整個中油運作的實例！立法院要跟中油要資料；中油卻以中油裡的商業機密為由不給立法委員監督；從每一次的理監事會議記錄可以看出端倪！這些批著國民黨中常委的理監事長、理監事丈著中油工會之名卻對著立法院對抗！這些人真的有照顧到員工權益嗎？為什麼這些人可以從國民黨中常委空降？ : 以下引用一篇經濟日報的社論並highlight 內容： : 早自民國87年經濟部即開始編列中油民營化預算，截至目前釋股預算已編列55.19%，約新台幣1,700億元，釋股對象包括全民、員工及特定投資人。但是，立法院院會在92年作成決議，中油民營化必須先取得工會同意，因為中油工會長期反對，以及主政者將公營事業員工視為選舉鐵票，中油民營化事實上已淪為政府大幅調漲油價時虛晃一招的道具。這樣的心態若不根本改變，民營化只是空談而已。 : 另一方面，中油本身原本也有提升經營績效的長程計畫，就是和民間業者共同組成國光石化公司，中油持股43%，是最大股東，意圖以民營事業型態和台塑化競爭。但是，國光石化開發案在去年受選舉因素影響而終止，中油金蟬脫殼之計宣告失敗。接下來五輕若在104年遷廠，中油產能缺口將大增，未來更難和台塑化競爭。因為國光石化政策的轉折，中油民營化的必要性顯得更高，包括五輕遷廠、石化業高值化計畫等相關問題的同步解決，也已刻不容緩。 : 再者，推動民營化不僅要有決心，而且必須有完整配套。過去民營化受到詬病，主因部分國營事業藉民營化逃避國會監督，但實質上仍是公股掌控的事業，如中華電信、中鋼及多家公股掌控的民營金融機構。依中油釋股計畫，政府持股45%，是單一最大股東，仍是受外界批評半吊子民營化，不符社會期待。此外，油品是寡占市場，中油民營化後不再擔負政策性任務，可能出現和台塑化聯合訂價、賺取暴利的情況，因此，在國營事業退出油品市場後，如何促進市場公平競爭，並合理規範油品訂價，亦是必須預先規劃的民營化配套。 : 和中油同屬能源事業的台電公司，也應該要有民營化的準備與規劃。台電負債累累原因固與電價長期偏低有關，但87年台電和民營電廠簽25年長期合約，每年花1,000多億元向民間高價購電，亦是虧損擴大的重要原因。國營事業受利益介入被當冤大頭，不僅要怪當年主事者缺乏擔當，而且更印證國營事業背後涉及的龐大利益糾葛。因此，推動台電民營化無疑是革除積弊的最有力途徑。另考量台電民營化牽涉至為複雜，短期內可先從引進民間資金著手，如目前台電鉅額虧損可以透過先減資再增資方式，引進民間投資，為邁向民營化跨出關鍵一步。 : 很多人期待馬總統任內至少要完成中油民營化，但二年後國內重要選舉陸續啟動，民營化很可能又成為禁忌。因此，中油民營化若要玩真的，政府須下定決心，並透過立法院取得朝野背書，讓民營化列車儘快上路，不再受政治因素羈絆。 : 資料來源： : http://ppt.cc/XcLR : 而中油工會現在已然變成了國民黨選舉的打手了！為什麼工會理事長選舉跑去給連勝文站台？ :
