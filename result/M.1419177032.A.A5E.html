※ [本文轉錄自 PublicIssue 看板 #1KbklJ69 ] 作者: q10242 (黑田祐司) 看板: PublicIssue 標題: [動員] 吳育昇選區吳育昇罷免案需要你的支持 時間: Sun Dec 21 23:48:27 2014 不是我在講，闌尾真的很難割。 在聯署的時候，民眾所填的資料裡面有姓名、身分證字號、性別、住址至少要填到里鄰、 還有簽名。 光是要填這些資料，就已經讓百分之十到二十想簽的民眾卻步不前了。 因為怕資料被拿去利用、怕資料會外洩、或者怕被抓去算帳。 接著，要填資料的時候，如果有任何簡體字，這一份聯署就不算。 如果有任何字超出格子，這一份聯署就不算。 如果有字寫得太潦草，這一份聯署也不算。 如果有任何塗改，也不算。 這些是規定，真的超嚴格的，拿來跟投票比較的話，民眾連抄資料都不需要，只要把章蓋 下去，就算是碰到邊也算有圈選到。不覺得選舉、和罷免實在是太不對等了嗎? 光是這樣就算了，還有更過分的。 之前的聯署書，和現在的聯署書格式改了，所以在隔式改了之後，先前收集的聯署書又全 部不算，又要全部重來。你說，這是不是中選會在搞我們? 不只中選會在搞我們，連郵局也在搞我們。我們曾經有志工進行一連串聯署書寄送的實驗 ，實驗證明，十封送出去的聯署書，只有四封會回到割闌尾辦公室，這實在是太扯。 就算聯署書沒有消失，有些聯署書在寄出去一個月後才送到辦公室，時效也已經過，就算 寄到也沒有用了。 接著是門檻問題，我不是說我不贊成罷免要有一個門檻，畢竟罷免案如果太容易成立，那 對手陣營如果選書就直接發動罷免就好了。 但是第一階段3%提案、第二階段15%人口聯署這個門檻也太困難了，再加上前面說要填的 資料，門檻真的比天高，有些人看15這個數字好像很小，可是在一個特定區域，15%是相 當多的。因為15%的人口幾乎是比當選的票數都還要高的。加上很多人不大關注時事，不 來簽的理由通常是因為不知道為什麼要罷免要罷免的對象。 (蔡正元是例外，他仇恨值超高所以聯署書收集很容易。) 如果只是15%，慢慢努力也還能收集，但是收集聯署書有時間限制，一個月內要收集15%的 聯署書，在我所在的地區，吳育昇選區也是蠻困難，因為15%代表三萬多人，而割闌尾的 人力真的有限，一個月內要收集三萬份的聯署書，超困難，雖然志工們包括我都咬著牙做 下去，但還是容許我在這靠北一下。 而且，在吳育昇選區這個地方還會有一個特有的現象，因為之前也有黃國昌老師進行罷免 的關係，有些人在那之前就簽過了，所以在收集聯署的時候難上加難，因為有人會認為自 己簽過了不能再簽，所以就算看到攤位、心中也想罷免吳育昇，卻不知道自己之前的簽名 已經無效了。 因此吳育昇選區罷免案是困難中的困難。加上吳育昇這個爛立委相當狡猾，都太不公開講 話，仇恨值超低，很少人知道他除了上薇格以外還幹了什麼骯髒事。 http://ppt.cc/1m2k 這是吳育昇的罷免理由書，裡面寫得超清楚。最讓我不能忍的是第一點、第二點和第六、 第十和十一點。不過老實說裡面寫的我通通都認為難以忍受。 最後是50%投票率這個門檻，台灣投票率還算高，但就算如此50%投票率的門檻真的很高， 就和公投一樣不投票等於反對的神邏輯一樣存在於罷免投票。而且罷免投票還不能和選舉 投票一起舉行，讓50%的門檻更加的高不可攀。 所以，在這呼籲吳育昇選區的朋友，我們還缺很多聯署書，只要是一個月前簽過的，都不 算，因為很有可能你所簽的那份是送出去但是沒有用掉的。 不要因為你簽過了就以為你有被算在裡面，答案是不一定，保險起見，只要是一個月前有 簽過的，我們還是希望你再來簽一次。 也不要擔心資料會外洩，所有的資料會在顧立雄律師的見證之下銷毀。 在我寫這篇文的同時，收集聯署書的期限也只剩下四天了，真的還蠻緊急的，拜託大家了 。 在此提供割闌尾辦公室地址 淡水區中山路171號1樓 代收聯署書地點 http://appy.tw/receiving-places/ 聯署書下載 http://appy.tw/2014/05/download/ 填好之後，可以的話也希望盡量不要用寄的，直接送到辦公室或者代收地點 前面也說過，郵局會搞我們 另外填寫的時候請注意資料不能寫簡體字、不要太潦草，不要超出格子，也不要塗改 名字也要填正楷 否則寧可重寫一張 要不然中選會就會雞蛋裡挑骨頭不給過 最後也別忘了簽名 如果你有親朋好友住在選區 也希望你可以問一下是否簽了聯署書 順便和親朋好友問候一下 我不是屬於吳育昇選區的人，但是我住在淡水，目前也在擔任吳育昇選區的志工 我的八卦版權限在318的時候為了快速發文字數太短被水桶，目前無發文權限 希望有人能幫忙轉八卦版 感謝 -- ψDeathDeath 這是我的鋼彈 這是我的意志 啊幹 有什麼武器可以用 ┌────┐ 我最了解它! 我非去不可! 別死啊 ...... ┌─(0w0 )┐ ┌───┐┌────┐ ┌───┐┌────┐ ┌─ V) )┐ 不准去 => ╞ ( ) │ (0wξ )│ => ╞ ("0w0) │ (-wξ )│ ┌─ ＜ │┐ 你會死的! │ (﹀Υ﹀Υ ㄏへ )│ │ (﹀Υ﹀Υ ㄏへ )│ ┌───│┐ \(wQ ) │|￣￣￣￣￣|───┘ │|￣￣￣￣￣|───┘ -- ※ 發信站: 批踢踢實業坊(ptt.cc), 來自: 180.217.2.211 ※ 文章網址: http://www.ptt.cc/bbs/PublicIssue/M.1419176915.A.189.html ※ 發信站: 批踢踢實業坊(ptt.cc) ※ 轉錄者: nitt (111.255.26.146), 12/21/2014 23:50:31
