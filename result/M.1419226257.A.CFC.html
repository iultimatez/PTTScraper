留下立委空缺將於明年2月補選，身為台中市民的吾爾開希，決定以無黨籍身份參選，和國 : 民黨、民進黨候選人一決高下。 : 對於參選動機，吾爾開希接受BBC中文網訪問時表示，台灣深陷藍、綠惡斗的困局而停滯 : 不前，他多年來一直關心出現第三勢力的可能性，但個人始終未付諸行動。 : 這次「九合一」地方選舉中，無黨籍的柯文哲突破藍綠分野，高票當選台北市 : 長，讓他深受啟發和鼓舞，決定參選台中的立委補選。 : 吾爾開希指出，無黨無派的柯文哲大獲全勝，無疑是選民用選票懲罰了執政的國民黨，同 : 時也警告了民進黨，人民尚未恢復對民進黨的信心，民進黨只能被迫支持非民進黨人士。 : 他說，「台灣智庫」在選後所做民調也顯示，有64%的人對於國、民兩黨感到厭惡，願意 : 把票投給兩黨以外的候選人。 : 台灣民主2.0時代 : 吾爾開希認為，從「太陽花學運」到「九合一」地方選舉，台灣已經進入「民主2.0時代 : 」，公民不再相信代議政治、政黨政治，而是以行動直接參與政治。 : 他說，在這個新時代中，權力不再源於世家、派系或是藍綠分野，而是公民自主和理性的 : 力量主宰了一切，在這前所未見的政治環境中，第三勢力得以突圍而出。 : 吾爾開希強調，這讓他有了參選的動力。他說：「在歷史的關鍵時刻，挺身而出是光榮的 : ！」 : 截至目前，民進黨確定由台中市議員黃國書代表參選，國民黨可能由前台中市 : 副市長、前行政院副秘書長蕭家淇出馬角逐。面對可能的競爭對手，本身又無政黨奧援，吾爾開希自 : 評優勢何在？ : 吾爾開希表示，蕭家淇來自於被人民以選票否定的胡志強市政團隊和江宜樺行 : 政團隊，國民黨如果推出這樣的人選，代表國民黨是不長進的政黨，其腐朽的黨內文化令 : 人反感。吾爾開希說，在此情況下，他的勝算就更大了。 : 至於黃國書，吾爾開希說，他是一位令人尊敬的人物，更是一位好的市議員，應該繼續留 : 在台中市為市民服務。 : 「一人敵兩黨」 : 至於本身的優勢，吾爾開希表示，這次補選產生的立委只剩一年任期，其主要 : 的任務就是修憲，藍綠兩黨在修憲議題上各有盤算而不敢表態，他本身沒有政黨包袱， : 「一人敵兩黨」，可以針對深化民主的議題暢所欲言——例如改採內閣制、立 : 委選舉改採聯立制，以及降低投票年齡為18歲——來爭取人民的認同和支持。 : 吾爾開希說，兩岸關係是另一個他可以發揮「一人敵兩黨」的領域。 : 他指出，台灣面臨外部的挑戰主要來自中國大陸，在藍綠都害怕共產黨的情況 : 下，台灣的利益被犧牲了，習近平在台灣的影響力愈來愈大，十分荒謬。 : 吾爾開希認為，國民黨錯誤的兩岸政策，讓台灣在害怕和依賴中被北京當局所擺佈，這是 : 十分危險的。他來自中國大陸，對於中共有更為清楚和正確的認識，在兩岸關係的議題上 : ，他能夠捍衛台灣的立場，讓兩岸政策回歸理性。 : 吾爾開希說，台中市民自主性高，游離票多，他將透過網路和文宣傳單與選民進行有效溝 : 通，讓專家跌破眼鏡。 : 不過，吾爾開希也指出，由於他的大陸背景，台灣部分電子媒體考量到大陸市 : 場，對他的態度相對不夠友善。他說，大陸異議分子在台發言受限，是一件可怕的事，台 : 果和大陸相比較，一定有體制性和結構上的原因，這同樣也已經對大陸產生巨大啟發和示 : 範作用。 : 吾爾開希說，他在推特上推文說要參選，大陸的追蹤者們都興奮極了，直說大 : 陸通緝犯在台灣參選立委，被台灣社會平靜接受了，就連流亡中的朋友也眼紅了。 : 吾爾開希表示，身為流亡者，台灣接納了他，台灣是他的第二祖國，如今眼看藍綠惡斗扼 :
